❓ Quick Help

📂 Supported File Types
  • NGS: Paired FASTQ files (.fastq, .fq, .gz). Exactly two files required (R1 & R2).
  • Sanger: AB1 (.ab1), FASTA (.fasta, .fa), plain text (.seq, .txt). One or more files.
🧠 Analysis Modes
  • Reference‑Based NGS: Aligns reads to a selected genome, calls variants, and annotates them.
  • De Novo Assembly (Sanger): Assembles reads without a reference using Greedy, Semi‑Global, or De Bruijn algorithms.
  • Guided Assembly (Sanger): Aligns reads to a user‑provided reference and builds a consensus sequence.
⚙️ Key Parameters
  • Min Quality (Q): Phred quality threshold for trimming (Q20 = 1% error).
  • Window Size: Number of consecutive bases used to compute average quality when trimming.
  • Trim Adapters (NGS): Removes adapter sequences and low‑quality ends (Trimmomatic).
  • Direction (Sanger): Auto / Forward / Reverse – controls which strand is used for alignment.
💾 Saving & History

Every analysis is automatically saved. Use the History page to view past runs, download results, or re‑inspect reports.

For more details, visit the project GitHub repository.
