🧬 About QuickNGS

Version 2.0 – NGS & Sanger Analysis Suite

QuickNGS is a smart, browser‑based platform that automatically detects the type of sequencing data you upload (FASTQ → NGS, AB1/FASTA/seq → Sanger) and runs the appropriate analysis pipeline.

  • NGS (Reference‑Based): FastQC → Trimmomatic → BWA → Samtools → FreeBayes → SnpEff → Clinical report
  • Sanger (De Novo): Greedy OLC / Semi‑Global Alignment / De Bruijn Graph assembly
  • Sanger (Guided): Align to reference with BWA, generate consensus, report variants

All analyses are saved automatically and can be reviewed at any time from the History page.
