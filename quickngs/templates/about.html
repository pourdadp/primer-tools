🧬 About QuickNGS

Version 2.0 – NGS & Sanger Analysis Suite

QuickNGS is a smart, browser‑based platform that automatically detects the type of sequencing data you upload (FASTQ → NGS, AB1/FASTA/seq → Sanger) and runs the appropriate analysis pipeline.

🔧 Tools Used
  • NGS Pipeline: FastQC, Trimmomatic, BWA, Samtools, FreeBayes, SnpEff
  • De Novo Assembly: Greedy OLC, Semi‑Global Alignment, De Bruijn Graph
  • Guided Assembly: BWA + Samtools + BCFtools consensus
  • Frontend: Bootstrap 5, Chart.js, JavaScript (Vanilla)
  • Backend: Python, Flask, Biopython, PyYAML, Plotly
📂 Part of Primer Tools Suite

QuickNGS works seamlessly with Primer Database Manager (PDM) and Sequencing Panel Optimizer (SPO).
